## Common Utility Methods

```java
// DimensionUtils.java
public class DimensionUtils {
    public static int dpToPx(Context context, float dp) {
        return (int) (dp * context.getResources().getDisplayMetrics().density);
    }

    public static float pxToDp(Context context, int px) {
        return px / context.getResources().getDisplayMetrics().density;
    }

    public static int spToPx(Context context, float sp) {
        return (int) (sp * context.getResources().getDisplayMetrics().scaledDensity);
    }

    public static boolean isTablet(Context context) {
        return context.getResources().getConfiguration().screenWidthDp >= 600;
    }

    public static boolean isWideScreen(Context context) {
        return context.getResources().getConfiguration().screenWidthDp >= 840;
    }
}
```

```java
// ViewUtils.java
public class ViewUtils {
    public static void fadeIn(View view, long duration) {
        view.setAlpha(0f);
        view.setVisibility(View.VISIBLE);
        view.animate().alpha(1f).setDuration(duration)
            .setInterpolator(new DecelerateInterpolator()).start();
    }

    public static void fadeOut(View view, long duration) {
        view.animate().alpha(0f).setDuration(duration)
            .withEndAction(() -> view.setVisibility(View.GONE)).start();
    }

    public static void slideInFromBottom(View view, long duration) {
        view.setTranslationY(100f);
        view.setAlpha(0f);
        view.setVisibility(View.VISIBLE);
        view.animate()
            .translationY(0f).alpha(1f)
            .setDuration(duration)
            .setInterpolator(new DecelerateInterpolator(2f))
            .start();
    }

    public static void staggerIn(List<View> views, long itemDelay) {
        for (int i = 0; i < views.size(); i++) {
            View v = views.get(i);
            v.setAlpha(0f);
            v.setTranslationY(40f);
            v.animate()
                .alpha(1f).translationY(0f)
                .setDuration(300)
                .setStartDelay(i * itemDelay)
                .setInterpolator(new DecelerateInterpolator())
                .start();
        }
    }

    public static void setCustomRipple(View view, int color) {
        RippleDrawable ripple = new RippleDrawable(
            ColorStateList.valueOf(color),
            view.getBackground(),
            null);
        view.setBackground(ripple);
    }
}
```

// ColorUtils.java
public class ColorUtils {
    public static int withAlpha(int color, float alpha) {
        int a = (int) (alpha * 255);
        return Color.argb(a, Color.red(color), Color.green(color), Color.blue(color));
    }

    public static int blendColors(int from, int to, float ratio) {
        return (Integer) new ArgbEvaluator().evaluate(ratio, from, to);
    }

    public static boolean isDarkColor(int color) {
        double luminance = 0.299 * Color.red(color) / 255
            + 0.587 * Color.green(color) / 255
            + 0.114 * Color.blue(color) / 255;
        return luminance < 0.5;
    }

    public static int getContrastColor(int background) {
        return isDarkColor(background) ? Color.WHITE : Color.BLACK;
    }
}

## Edge-to-Edge Setup (recommended for all apps)

### `MyApplication.java`
```java
public class MyApplication extends Application {
    @Override
    public void onCreate() {
        super.onCreate();
        // Apply dynamic colors to all activities
        DynamicColors.applyToActivitiesIfAvailable(this);
    }
}
```

### Base Activity pattern
```java
public abstract class BaseActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);

        // Edge-to-edge
        WindowCompat.setDecorFitsSystemWindows(getWindow(), false);

        // Transparent bars
        getWindow().setStatusBarColor(Color.TRANSPARENT);
        getWindow().setNavigationBarColor(Color.TRANSPARENT);
    }

    protected void applyInsets(View rootView) {
        ViewCompat.setOnApplyWindowInsetsListener(rootView, (v, insets) -> {
            Insets bars = insets.getInsets(WindowInsetsCompat.Type.systemBars());
            v.setPadding(bars.left, bars.top, bars.right, bars.bottom);
            return WindowInsetsCompat.CONSUMED;
        });
    }

    protected int dpToPx(float dp) {
        return (int) (dp * getResources().getDisplayMetrics().density);
    }
}
```

## Performance Checklist

### RecyclerView
- [ ] Use `DiffUtil` — never `notifyDataSetChanged()`
- [ ] `setHasFixedSize(true)` when item dimensions are static
- [ ] `setRecycledViewPool()` for shared nested RecyclerViews
- [ ] Use `ViewHolder` pattern strictly — no `findViewById` in `onBind`
- [ ] Load images asynchronously (Glide/Coil)
- [ ] Use `MaterialDividerItemDecoration` instead of view-based dividers

### Layout
- [ ] Flat ConstraintLayout > nested LinearLayouts
- [ ] Use `ViewStub` for infrequent UI (empty states, error banners)
- [ ] Use `merge` tags in reusable layouts
- [ ] Minimize overdraw — check with `Show GPU Overdraw`

### Images
- [ ] Use `WebP` format for photos, `VectorDrawable` for icons
- [ ] Set explicit `layout_width/height` on `ImageView` to avoid layout thrash
- [ ] Use `centerCrop` or `fitXY` — never `matrix` unless needed
- [ ] Glide `diskCacheStrategy(ALL)` for network images

### Animations
- [ ] Hardware layer (`LAYER_TYPE_HARDWARE`) during complex animations
- [ ] Cancel animators in `onStop`/`onDestroy`
- [ ] Prefer `ViewPropertyAnimator` over `ObjectAnimator` for simple moves
- [ ] Use `MotionLayout` instead of manual animator chains for complex scenes
- [ ] Lottie: `setRenderMode(RenderMode.HARDWARE)` for complex animations

### Threading
- [ ] Never do I/O on main thread
- [ ] Use `ViewModel + LiveData` or coroutines for data loading
- [ ] Post UI updates via `runOnUiThread` or `Handler(Looper.getMainLooper())`

## Accessibility Checklist

- [ ] All interactive views have `android:contentDescription`
- [ ] Touch targets ≥ 48×48dp
- [ ] Color contrast ≥ 4.5:1 (text), ≥ 3:1 (large text/icons)
- [ ] Focus order matches visual reading order
- [ ] No information conveyed by color alone
- [ ] `android:importantForAccessibility="no"` on decorative images
- [ ] Tested with TalkBack enabled
- [ ] `android:accessibilityLiveRegion="polite"` on dynamic text areas
- [ ] `MaterialSwitch` always has a descriptive label
- [ ] `TextInputLayout` `hint` is always set (becomes label)
- [ ] Custom views implement `AccessibilityNodeInfo`
